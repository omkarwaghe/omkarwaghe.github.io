topics under CI

  • Use keys
    • Get keys - These key fields are automatically map to search key fields
    • Find keys - this maps to both search keys and alternate search keys for underlying component
    • create keys - If underlying component allows add action, then create keys are generated for the component interface automatically
    Theses keys are created automatically when you create a component interface. you can modify get key if any search key is added or find keys if you want to restrict user to not search with any alternate search key fields.
